# FIFA 世界杯数据可视化 — 项目架构设计

> 课设项目 · 数据可视化 · Vue3 + ECharts5  
> 最后更新：2026-06-09 v3（FIFA品牌深蓝+金色主题）

---

## 一、项目概述

| 项目名 | FIFA World Cup Data Visualization |
|--------|--------------------------------|
| 主题 | 世界杯体育赛事数据可视化（1930–2026） |
| 亮点 | 历史数据 + 2026今届实时元素 双轨叙事 |
| 截止日期 | 2026-06-19（世界杯小组赛进行中） |

---

## 二、技术架构

### 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | Vue 3 + Vite | 热更新快，适合课设周期 |
| 可视化库 | ECharts 5 + vue-echarts | 地图/桑基图/雷达图/热力图 成熟 |
| UI 组件 | 自研组件体系 | 无需 Element Plus，自行实现更轻 |
| 页面切换 | Tab 组件内切换（非路由） | 参考直播吧/懂球帝 Tab 导航模式 |
| 国旗渲染 | SVG 国旗图标 | flagcdn.com 实时加载 |
| 数据层 | 静态 JSON + 可扩展 API | 当前静态，预留动态接口 |
| 部署 | GitHub Pages / Vercel | 免费静态托管，答辩演示方便 |

## 五、动画体系（v3 新增）

### 进入动画

| 类名 | 效果 | 用途 |
|------|------|------|
| `.anim-fadeInUp` | 从下方24px淡入 | 卡片、列表项默认 |
| `.anim-fadeIn` | 纯淡入 | 轻量元素 |
| `.delay-1` ~ `.delay-8` | 0.1s~0.8s延迟 | 交错出现 |

### 交互动画

| 触发 | 效果 | 说明 |
|------|------|------|
| 卡片 hover | `translateY(-4px)` + 阴影加深 | 通用卡片上浮 |
| 快捷入口 hover | `scale(1.1) rotate(-4deg)` | 图标弹跳 |
| 冠军列表 hover | `translateX(4px)` | 右滑强调 |
| Tab 切换 | `fade + translateY(10px)` | Vue Transition |
| Tab 指示线 | `scaleX(0→1)` | 下划线动画 |

### 持续动画

| 类名 | 效果 | 用途 |
|------|------|------|
| `.anim-pulse` | `scale(1→1.03)` 2s循环 | 2026徽章 |
| `.anim-float` | `translateY(0→-8px)` 3s循环 | 装饰元素 |
| `.anim-glow` | 金色阴影脉冲 | 强调卡片 |
| 倒计时数字变化 | `scale(1.2) + 金色闪烁` | 秒数跳动 |
| 数据卡数字 | `0→目标值` 1.2s缓动 | 数字增长 |
