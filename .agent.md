# worldcup-viz Agent

这是 `worldcup-viz` 项目的专用 Copilot agent，适合处理本仓库内的前端开发、数据可视化、组件布局与项目配置任务。

## 适用场景

- 修改或调试 `src/` 中的 Vue 3 单文件组件 (`.vue`)
- 优化 Vite 配置、修复构建/预览问题
- 开发 ECharts 图表、数据展示和交互样式
- 调整 Element Plus 组件、布局与响应式页面
- 处理 JSON 数据加载、数据映射和可视化数据逻辑
- 进行项目级重构、性能优化和功能增强

## 优势

- 明确聚焦于本项目的技术栈：Vue 3、Vite、ECharts、Element Plus
- 避免默认 agent 的过于泛化建议
- 方便在本仓库中选择“项目专用助手”时获得更精准的上下文感知

## 如何使用

当你在本仓库中提问时，可直接选择或指定该 agent。如果你的聊天环境支持 agent 列表，优先选中 `worldcup-viz-agent`。

示例提示：
- “请帮我修复 `src/components/ChartCard.vue` 的 ECharts 配置问题。”
- “这个 Vue 组件为什么在 `npm run dev` 时报错？”
- “我想把 `src/tabs/TabOverview.vue` 里的数据显示改成可筛选格式。”

## 备注

本文件目前存放在仓库根目录，已足够用于本地工作区。如果你希望让团队共享此 agent，可以将文件复制到 `.github/agents/worldcup-viz.agent.md`。
